Comandos Básicos OLT ZTE.

Neste artigo veremos os comandos mais comuns para o dia a dia de análise em uma OLT ZTE.

Os comandos citados abaixo foram validados para OLT ZTE ZXA10 Software, Version: V1.1.2_BC, Release software


Comandos Show
O comando ter len desabilita o limite de linhas na saída de um comando


ter len 0
O comando show clock exibe a hora e a data do sistema.

show clock
O comando show ip interface brief exibe um resumo das interfaces IP.

show ip interface brief

O comando show equip exibe informações sobre o hardware e o software de um dispositivo.
A saída do comando show equip pode variar dependendo do dispositivo e da versão do software. No entanto, geralmente inclui informações como:
– O nome do dispositivo
– O tipo de dispositivo
– O número de série
– A versão do software
– As informações de configuração
– Os recursos disponíveis
O comando show equip é útil para diagnosticar problemas com um dispositivo ou para obter informações sobre as especificações de um dispositivo.

show equip

O comando show card-clock exibe a hora e a data do relógio do cartão.
O relógio do cartão é um relógio independente que pode ser usado para manter a hora e a data de um dispositivo, mesmo que o relógio do sistema esteja desligado. O comando show card-clock é útil para verificar se o relógio do cartão está funcionando corretamente.

show card-clock
O comando show install pode ser usado para exibir informações sobre o status de uma instalação de software ou firmware em um dispositivo. A saída do comando pode incluir informações como:
– O status da instalação
– O progresso da instalação
– Os erros ou avisos da instalação
O comando show install é útil para monitorar o progresso de uma instalação ou para diagnosticar problemas com uma instalação.


show install

show install commit
